# 阳痿次新（YWCX）评分机制说明

> 风险声明：本模型仅用于学习研究与辅助决策，不构成任何投资建议；请自行承担交易风险。

本文解释“**阳痿次新（YWCX）**”评分模型的计算逻辑，以及未来导出股票池时的筛选口径。

> 模型定位说明：
> **YWCX 是一个“极弱 → 刚苏醒”的次新低位模型**，不追求确定性主升浪，而是捕捉
> *被市场冷落、流动性枯竭、刚出现最小级别趋势修复* 的早期状态。

---

## 1. 总体流程（从日线到评分）

1. 拉取并入库日线 OHLCV → `stock_daily`
2. 计算基础技术指标 → `stock_indicators`

   * MA5 / MA10 / MA20
   * 成交量均量（5 / 10 / 20）
   * ATR14
3. 计算新股相关统计 → `stock_ipo_stats`

   * 上市以来最高价 / 最低价
   * 是否破发
4. 计算 YWCX 评分与标签 → `stock_scores_ywcx`

---

## 2. 硬性样本过滤（先执行）

**YWCX 是“强前置约束模型”**，不满足以下条件的股票 **直接不进入评分阶段**。

### 2.1 次新时间窗过滤（Mandatory）

仅保留：

```
上市交易日数 ≤ 60 个交易日
```

> 直觉：
> 超过 60 日的票，筹码结构与心理结构已不再是“次新”。

---

### 2.2 爆炒否决（Overhype Filter）

上市以来若出现任一情况，直接否决：

1. 任一交易日：

```
(high / issue_price) ≥ 1.8
```

2. 或上市后任意 5 日累计涨幅 ≥ 60%

> 直觉：
> **被爆炒过的次新，已经“硬过一次”，不属于阳痿样本。**

---

## 3. 风险否决（评分前执行）

命中任一条，当日评分直接归零，并打上 `RISK_FILTERED`。

### 3.1 暴力反转否决（Violent Reversal）

在最近 5 个交易日内：

* 单日涨幅 ≥ 9%
* 或 3 日累计涨幅 ≥ 20%
* 或 放量暴拉：

  ```
  volume ≥ 2.0 × 10日均量
  ```

> 直觉：
> YWCX 要的是“刚醒”，不是“被人一脚踹醒”。

---

### 3.2 高位偏离否决（Relative High Filter）

计算当前价相对上市以来最高价的距离：

```
distance = (ipo_high - close) / ipo_high
```

若：

```
distance < 0.25
```

则否决。

> 直觉：
> 阳痿模型只做 **接近地板的东西**。

---

## 4. 子分项评分（每项 0～10）

未触发否决时，计算 **6 个子分项**，全部围绕“弱、冷、刚转向”。

---

## 4.1 位置衰弱分（weak_position_score）

衡量是否 **破发 + 靠近新低**。

### 计算项：

* 是否破发：

  ```
  close < issue_price
  ```
* 距上市以来最低价：

  ```
  (close - ipo_low) / ipo_low
  ```

### 打分：

| 条件             | 分数 |
| -------------- | -- |
| 破发 且 距新低 ≤ 10% | 10 |
| 破发 且 距新低 ≤ 20% | 6  |
| 未破发            | 0  |

---

## 4.2 缩量枯竭分（volume_dry_score）

只关注 **最近 5～10 日成交量趋势**。

### 条件判断：

1. 最近 5 日均量 < 最近 10 日均量
2. 最近 5 日均量 ≤ 最近 20 日均量 × 0.7

### 打分：

| 条件         | 分数 |
| ---------- | -- |
| 同时满足 1 + 2 | 10 |
| 仅满足其一      | 6  |
| 否则         | 0  |

> 直觉：
> **没人交易 ≠ 坏事**，在次新里往往是“卖无可卖”。

---

## 4.3 极弱波动分（low_volatility_score）

用 ATR 判断是否“缩到不能再缩”。

```
atr_ratio = ATR14 / close
```

| atr_ratio | 分数 |
| --------- | -- |
| ≤ 2.5%    | 10 |
| ≤ 4%      | 6  |
| > 4%      | 0  |

---

## 4.4 微趋势反转分（micro_trend_score）

**这是核心分项之一，但刻意压制分值。**

### 前置条件：

* 之前至少 5 日：

  ```
  close < MA5
  ```

### 当前满足：

* 当日：

  ```
  close > MA5
  ```
* 但：

  ```
  close ≤ MA10 × 1.03
  ```

### 打分：

| 条件       | 分数 |
| -------- | -- |
| 满足全部条件   | 10 |
| 仅刚站上 MA5 | 6  |
| 否则       | 0  |

> 直觉：
> **刚翻身坐起来，还没站稳。**

---

## 4.5 流通市值分（float_cap_score）

仅使用 **实际流通市值**（单位：亿元）。

| 流通市值      | 分数 |
| --------- | -- |
| ≤ 30      | 10 |
| 30～60     | 6  |
| 60～100    | 3  |
| >100 / 缺失 | 0  |

> 直觉：
> **YWCX 的弹性几乎全部来自“小”。**

---

## 4.6 冷关注度分（attention_score）

衡量“市场是否已经彻底不看它了”。

### 条件（近 10 日）：

* 涨幅 > 3% 的交易日 ≤ 1 天
* 任一日成交额未显著放大（≤ 20 日均额 × 1.3）

| 条件    | 分数 |
| ----- | -- |
| 同时满足  | 10 |
| 仅满足其一 | 6  |
| 否则    | 0  |

---

## 5. 总分合成（0～100）

```
total_score =
  weak_position_score   * 20%
+ volume_dry_score      * 20%
+ low_volatility_score  * 15%
+ micro_trend_score     * 20%
+ float_cap_score       * 15%
+ attention_score       * 10%
```

> 说明：
> **趋势权重被刻意压制**，位置 + 枯竭是第一优先级。

---

## 6. 标签（status_tags）

当未触发否决时，根据关键状态打标签：

* `BROKEN_IPO`：破发
* `NEAR_IPO_LOW`：距上市低点 ≤ 15%
* `VOLUME_DRY`：缩量成立
* `LOW_VOL`：ATR 极低
* `JUST_ABOVE_MA5`：刚站上 MA5
* `SMALL_FLOAT`：流通市值 ≤ 30 亿

若触发否决，仅保留：

```
["RISK_FILTERED"]
```

---

## 7. 模型直觉总结（非代码）

**YWCX 本质不是“做强”，而是：**

* 做 **没人要的**
* 做 **跌无可跌的**
* 做 **刚刚停止恶化的**
* 做 **小而轻、稍微给点量就能动的**

它是 **LAOWANG v3 的时间前置版本**，
也是 **主升浪之前 1～2 个心理阶段的观察池模型**。
